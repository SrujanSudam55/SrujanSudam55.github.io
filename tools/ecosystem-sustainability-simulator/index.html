Ecosystem Sustainability Simulator

A constraint-based simulator for ecosystems where agents compete for shared resources, consumption follows priority rules, and depletion is irreversible. The point is feasibility, not optimization.

View the repo on GitHub Back to tools

What it does

  • Enumerates combinations Evaluates all valid subsets and checks which ones are globally sustainable.
  • Simulates sequential consumption Applies greedy priority-based consumption rules and updates state after each action.
  • Detects failure conditions Flags unmet needs and full resource depletion as unsustainable by design.

This project is about translating narrative rules into executable logic and showing where intuition fails in dynamic systems.

Quick run

  • Install pip install pandas openpyxl
  • Run python solve_ecosystem.py
  • Input and output Uses input.xlsx and exports results to solve_output.xlsx for inspection.
